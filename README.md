# AI Study Notes Summarizer

AI Study Notes Summarizer is a frontend web application built using HTML, CSS, and JavaScript. It helps students generate summarized notes and quiz questions from study content through a clean and responsive UI.

## Features
- Generate summarized notes
- Create quiz questions
- Modern UI
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript
- Font Awesome
